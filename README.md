dutyplanner
===========

Tool for plan duty (experiment for learning Haskell)

1. Install stack (http://hackage.haskell.org/package/stack)
2. Run

 stack setup

for install haskell environment
3. Run

 stack build
 stack install

for build project
3. Run

 ~/.local/bin/DutyPlanner 2015-09-06 2015-09-26 persons.txt
